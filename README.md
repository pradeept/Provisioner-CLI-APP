# R Studio Provisioner

__Cli app to provision a R Studio on a remote server.__

## Tech Stack 🛠️
 ```
 - Node.js
 - Chalk.js
 - SSH2
 - Figlet
 ...
```

## Setup and Run ⚙️

``npm i``

``nodemon index.js``

## Example env 🔑
```
HOST=192.168.2.1
HOST_PORT=22
SSH_USERNAME=admin
SSH_PASSWORD=strongpassword
```

## Pre requisites on remote server 📢
- Docker
- Firewall Rules
